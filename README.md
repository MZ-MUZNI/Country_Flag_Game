# Country Flag Game

A mobile Android application built with Kotlin that tests users' knowledge about world flags and countries. The game features multiple difficulty levels and an exciting countdown mode challenge.

## Overview

Country Flag Game is an educational and entertaining mobile application where users can learn and test their knowledge about flags from different countries around the world. The app provides various game modes and difficulty levels to keep the gameplay engaging and challenging.

## Features

- **Multiple Game Modes**: Different levels of difficulty to suit all ages and knowledge levels
- **Flag Recognition**: Identify countries based on their flags
- **Country Name Guessing**: Guess the correct country name for displayed flags
- **Advanced Levels**: Challenge yourself with multiple flags and faster time limits
- **Countdown Timer Mode**: Compete against the clock for a more thrilling experience
- **Score Tracking**: Monitor your progress and achievements
- **Interactive UI**: User-friendly interface built with Jetpack Compose
- **Responsive Design**: Dynamic color scheme with system theme support

## Technologies

- **Language**: Kotlin
- **Frontend Framework**: Jetpack Compose
- **Minimum SDK**: Android 12+
- **Build System**: Gradle (Kotlin DSL)
- **UI Components**: Material 3 Design System

## Getting Started

### Prerequisites

- Android Studio (Latest version)
- JDK 21+
- Android SDK 12 or higher

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MZ-MUZNI/Country_Flag_Game.git
```

2. Navigate to the project directory:
```bash
cd Country_Flag_Game/MobileCoursework
```

3. Open in Android Studio and build the project

4. Run on emulator or device

## How to Play

1. **Launch the App**: Open the application to see the main menu
2. **Select Difficulty**: Choose from available difficulty levels
3. **Select Game Mode**: 
   - Normal mode: Take your time to answer
   - Countdown mode: Race against the clock
4. **Guess the Flag**: Select the correct country name
5. **Check Results**: View your score and correct answers
